# flappy-bird-app
 
## Problema
_Se necesita generar una captura de pantalla cuando el personaje cruza las columnas, guardar dicha captura en formato de imagen y en un archivo local._

---

## Requisitos
- Detectar cuando el personaje (_Bird_) atraviesa un set de columnas, como lo hace para contar el puntaje.
- En el momento que se obtiene un "punto" se debería capturar la pantalla.
- Esta captura no debe bloquear el flujo de juego (la aplicación no debería congelarse).
- Posterior a la captura de la pantalla, esta debería ser escrita en disco en formato jpeg, png, bmp o el formato que considere apropiado. Esta operacion tampoco deberia congelar la aplicación.
## Proceso
- Investigar métodos posibles para resolver la problemática planteada.
- Plantear cual seria el flujo a seguir.
- Justificar la decisión tomada para resolver la situación.
